--------raspicam------------ (Används inte längre!)

1. ladda ner https://sourceforge.net/projects/raspicam/files/?
2. flytta över filerna till pi'en
3.1 kompilera
  	tar xvzf raspicamxx.tgz
	cd raspicamxx
	mkdir build
	cd build
	cmake ..

3.2 installera
	make
	sudo make install
	sudo ldconfig

---------bcm2835------------
1. ladda ner wget http://www.airspayce.com/mikem/bcm2835/bcm2835-1.36.tar.gz
2. kompilera 
	tar zxvf bcm2835-1.xx.tar.gz
	cd bcm2835-1.xx
	./configure
	make
	sudo make check
	sudo make install

--------wiringpi------------
1. ladda ner https://git.drogon.net/?p=wiringPi;a=summary ta senaste och klicka på snapshot
2. flytta över filen till pi'en
3. kompilera
	tar xfz wiringPi-xxxxxx.tar.gz
	cd wiringPi-xxxxxx
	./build

-------Userland------------- (behövs för nya kamera api
1. Flytta över zipfilen ifrån mappen API till pi'en.
2. Lägg .zip filen i /opt/vc/ och packa upp, du ska nu få en mapp som heter "userland-master"
3. cd userland-master
4. sudo mkdir build
5. cd build
6. sudo cmake -DCMAKE_BUILDTYPE=Release ..
7. sudo make (tar ca 1h)
8. sudo make install
